# LibreMiddleware

Qt-free C++20 static libraries for reading Serbian smart cards (eID, vehicle registration, health insurance) and generic CardEdge/PKCS#15 tokens via direct PC/SC APDU communication.

Includes a PKCS#11 shared library for browser integration (Firefox, Thunderbird).

**Documentation & downloads:** [librescrs.github.io](https://librescrs.github.io)
